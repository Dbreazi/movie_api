Strobe API Documentation

Welcome to the Strobe API documentation! This API is designed for the Strobe application, a comprehensive platform dedicated to documentary sports. Strobe allows users to explore, manage, and enjoy a vast collection of sports documentaries, providing detailed information and media content about various sports events and personalities.

Endpoints

/users

POST /users

Description: Create a new user

Request Body:
{
  "name": "string"
}

Response:

Status: 201 Created
{
  "id": "string",
  "name": "string",
  "favouriteMovies": []
}
    
Status: 400 Bad Request
"users need names"

PUT /users/:id

Description: Update a user by ID

Request Parameters:
{
  "id": "number"
}
Request Body:
{
  "name": "string"
}

Response:

Status: 200 OK
{
  "id": "number",
  "name": "string",
  "favouriteMovies": ["array"]
}
    
Status: 400 Bad Request
"No such user"

POST /users/:id/:movieTitle

Description: Add a favorite movie for a user by ID

Request Parameters:
{
  "id": "number",
  "movieTitle": "string"
}

Response:

Status: 200 OK
"${movieTitle} has been added to user ${id}'s array"
    
Status: 400 Bad Request
"No such user"

DELETE /users/:id/:movieTitle

Description: Remove a favorite movie for a user by ID

Request Parameters:
{
  "id": "number",
  "movieTitle": "string"
}

Response:

Status: 200 OK
"${movieTitle} has been removed from ${id}'s array"
    
Status: 400 Bad Request
"No such user"

DELETE /users/:id

Description: Delete a user by ID

Request Parameters:
{
  "id": "number"
}

Response:

Status: 200 OK
"user ${id} has been deleted"
    
Status: 400 Bad Request
"No such user"
